This directory (maqueta/) is a master template — not a live project. It contains:

- **docs/_templates/**: 4 foundation doc templates (design-brief, content-brief, page-plans, motion-spec)
- **prompts/**: 17 prompts organized by phase (discovery -> identity -> content -> architecture -> motion -> 3D -> implementation -> quality)
- **inspiration/**: Reference catalog (sites by rubro, UI patterns, motion techniques, 3D references)
- **guides/**: Process documentation (project init, pipeline, design-first, Pegasuz, quality, delivery)
- **_project-scaffold/**: Copy-ready Vue 3 + Vite + Pinia starter with tokens.css and api.js
- **.claude/**: Agents (7), skills (19), hooks, settings — all generic, not client-specific

**Why:** Every new project starts from here. Copy the structure, fill the docs, then build.

**How to apply:** When Mateo says "nuevo proyecto", use guides/00-project-init.md as the starting point. Each project gets its own unique identity — never reuse palettes, typography, or motion personality.

# Curated Seed Library

`_components/` is a library of curated creative anchors for the front-brain.

## What these are

- Seeds, not final templates.
- Opinionated starting points with strong compositional bias.
- Mutation-ready components constrained by an explicit manifest.

## Source of truth

- `blueprints.manifest.js` defines problem solved, motion family, mutation budget, locked zones, mutable zones, required assets, and anti-pattern coverage for every seed.

## Rule

If a seed is reused as a rigid template, it failed. The goal is guided originality, not copy-paste output.

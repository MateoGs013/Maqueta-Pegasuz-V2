## Que observar

- **Personalidad:** el portfolio ES la persona/marca
- **Trabajo primero:** los proyectos son protagonistas
- **Navegacion experimental:** puede romper convenciones
- **Motion identitaria:** el movimiento define la personalidad
- **Minimalismo funcional:** menos UI, mas impacto visual

## Patrones comunes

| Pattern | Descripcion | Cuando usar |
|---------|------------|------------|
| Project list | Lista de proyectos con hover preview | Portfolio minimal |
| Project grid | Grid masonry o uniforme | Portfolio visual |
| Case study long-scroll | Pagina larga contando el caso | Portfolio detallado |
| Image sequence on scroll | Imagenes que cambian con scroll | Portfolio cinematico |
| Custom cursor contextual | Cursor que muestra preview | Portfolio interactivo |
| Name as hero | Nombre/marca como elemento hero tipografico | Personal brand |

## Tecnicas de diferenciacion

1. **La navegacion ES la experiencia** — No un menu generico
2. **Transiciones entre proyectos** — Flujo continuo, no cortes
3. **Tipografia extrema** — Tamaños enormes, fonts experimentales
4. **Monochrome + accent** — Una paleta personal reconocible
5. **Detalle obsesivo** — Cada pixel tiene proposito

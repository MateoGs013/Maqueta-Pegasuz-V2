## Que observar en este rubro

- **Fotografia:** plating como arte, lighting cinematico, closeups texturales
- **Color:** paletas inspiradas en ingredientes (earth tones, especias, frescos)
- **Tipografia:** serif elegante para fine dining, display bold para casual
- **Menu:** presentacion como experiencia de lectura, no PDF
- **Reservas:** CTA de reserva siempre visible
- **Atmosfera:** el sitio debe transmitir como se siente estar en el restaurante

## Patrones comunes

| Pattern | Descripcion | Cuando usar |
|---------|------------|------------|
| Fullscreen plate hero | Foto de plato a pantalla completa | Homepage fine dining |
| Interactive menu | Menu filtrable con fotos | Menu page |
| Chef/team story | Storytelling del chef | About section |
| Reservation CTA sticky | Boton de reserva flotante | Global |
| Gallery food grid | Masonry de fotos de platos | Gallery page |
| Ingredient journey | Scroll-driven historia del ingrediente | Storytelling |

## Tecnicas de diferenciacion

1. **Sinestesia digital** — evocar sabor y aroma a traves del visual
2. **Menu como experiencia** — no un PDF, sino diseno interactivo
3. **Ingrediente como hero** — contar la historia de UN ingrediente
4. **Photography editorial** — cada foto como tapa de Bon Appetit
5. **Cultural identity** — la identidad cultural como diferenciador visual

# Inspiration System

Sistema organizado de referencias visuales, patrones UI, tecnicas de motion, y elementos 3D.
Usar como biblioteca de ideas al comenzar cada proyecto nuevo.

## Como usar

1. **Antes de disenar:** revisar `sites/` para encontrar referencias del rubro
2. **Al planificar secciones:** revisar `patterns/` para layouts probados
3. **Al definir motion:** revisar `motion/` para tecnicas de animacion
4. **Al definir 3D:** revisar `3d/` para tecnicas WebGL

## Estructura

```
inspiration/
  sites/          Referencias de sitios completos por rubro
  patterns/       Patrones UI reutilizables por componente
  motion/         Tecnicas de animacion por tipo
  3d/             Tecnicas WebGL/Three.js por tipo
```

## Regla de oro

La inspiracion informa, no dicta. Cada proyecto debe tener identidad unica.
Tomar CONCEPTOS, no copiar EJECUCIONES.
